DDNS动态域名平台最佳选择：支持IPV6且提供免费子域名

# DDNS平台对比点评：
Dynv6.com
[23-01-28]【1】		 免费v6动态域名里名气最大的
+支持v6；免费域名
+API双栈；
ydns.io
[23-01-28]【2】		 +免费域名ydns.eu；支持v6
+控制台有IP历史记录，方便观测自家宽带多久IP变动一次
+API双栈：curl --ipv6可强制更新v6
desec.io
[23-01-28]【3】		 +免费域名15个dedyn.io；支持v6
+API有独立v6地址：https://update6.dedyn.io/
+API双栈：curl --ipv6可强制更新v6
goip.de
[23-01-27]		 +参数v6，短后缀goip.de、goip.it.
-API单栈v4，只能update参数中指定v6地址。
duckdns.org
[23-01-28]		 +免费域名
-API单栈v4（亚马逊VPC没有v6），必须参数指定v6.
dnsexit.com
[23-01-28]		 +免费域名、托管域名
-免费域名一年有效期、到期前30天内可以续期。
-API单栈v4，可参数指定v6提交
dynu.com
[23-01-28]		 +免费4个+托管4个，支持v6，
+双栈api.dynu.com、v6专用api-ipv6.dynu.com
-国内API网址被墙，v6也无法ping通，等于全废。
[DNS-O-Matic] 
[23-01-28]		 分发管理其他DDNS服务的平台，支持dynu
-API单栈v4；是否支持参数v6未测。
freedyn.de
[23-01-28]		 +API有独立v6地址、双栈API、子域名、托管域名
★ freedyn.de, ★ 2nk.de
-不支持Gmail邮箱。+支持163邮箱。
-注册测试使用1小时内就被删号。
freedns.afraid.org
[23-01-28]		 +免费5个、支持v6、有超短2两位数域名
-账号6个月未登录则休眠，域名会被停用。尝试自动登录脚本没用成功。
-API单栈v4
nsupdate.info +awsmppl.com, nerdpol.ovh, nsupdate.info
-据说：用api更新域名，不允许连续不停的更新。否则会被设置为abuse。同时还会收到警告邮件。
每周六的凌晨3点UTC,检查nochg的更新次数 >150 次,就会被设置为abuse,不能继续更新。需要手工登录网站去复位。
要求 ip 变化才更新，不变化不更新。(不变化时,只更新几次,没关系)
ChangeIP.com
[23-01-28]		 +域名后缀极多ddns.us
-API单栈v4，
-API不支持参数提交V6，只能进入web控制修改V6记录。
noip.com -据说：不是完全的免费，因为每个30天邮件激活一次
cloudns -据说：以前 cloudns 可管理或申请三个免费可解析的二级域名，改版后的免费额度只有一个了。只要不滥用，cloudns 的免费域名是很稳定的，但如果添加的解析记录失效，域名可能会被删除，当然你还可以重新申请回来，因为帐户仍然有效，这是使用者需要注意到的。
luadns.com
[23-01-29]		 免费限额Zones三个；Records三十个；
-不提供免费子域名，只能托管域名
dns.he.net
[23-01-29]		 -不提供子域名 
meibu.com
[23-01-29]		 -输入您的抖音号并加86745194305为好友(搜索每步动态域名解析)，否则只能用1周。
-3周内不登陆的用户，域名保留，但DNS解析记录要删除。
-半年内不使用的域名就要删除，释放以后其他人可以注册。
xname.org
[23-01-28]		 已死，网站前台能打开，后端数据连不上，什么也做不了。
# 获取本地互联网网关（公网）IP地址的方式：
  • 信息来源
    • qmcgaw/ddns-updater - Docker Image | Docker Hub
  • 识别IP服务平台：
    • ip.sb/    双栈;
    • test.ipw.cn/    双栈； # 测试网络是IPv4还是IPv6访问优先(访问IPv4/IPv6双栈站点，如果返回IPv6地址，则IPv6访问优先)
    • icanhazip.com/     双栈
    • ident.me/    双栈
    • https://www.nsupdate.info/myip     双栈 #必须加https前缀使用
    • ifconfig.io/ip    双栈
    • api6.ipify.org/    单栈v6；    单栈v4：api.ipify.org、api4.ipify.org；    双栈但只能获取v6：ipify.org
    • ip1.dynupdate6.no-ip.com/   单栈v6；
