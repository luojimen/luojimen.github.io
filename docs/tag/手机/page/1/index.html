手机改装无电池工作(法拉电容替代电池)[教程]

  • 文章作者 SSJ
  • 发布日期 2020年12月22日
  • 手机改装无电池工作(法拉电容替代电池)[教程]有1条评论
  • 电容选择注意点:由于开机电流较大,对电容放电能力有要求,因此不是什么电容都可以,像纽扣型电容放电能力较弱无法启动手机就不能胜任,需要选用峰值放电6瓦以上的组合型电容

实践设备:

  • 手机: 红米Note5 (2018年设备)
  • 电容: 法拉电容组合型5.5V 1.0F //可给手机亮屏续航4秒.
    https://item.taobao.com/item.htm?id=622910347656
  • 电源: 小米3A充电器(MDY-04-ES)
    • 可以使用任意5V2A以上电源适配器,例如5V2A的UPS或者手机充电器.
    • 5V1A的电源不行,手机自身高负荷运行时若是供电不足会使手机直接掉电.
  • 线材: USB数据线 5V3A (小米支持快充的数据线)
    • USB线不能太细,应当要能支持10W供电通过的USB线.一些赠送的线很可能达不到.若是出现意外掉电情况,检查电源和USB线是否足够承当负载.

步骤

  • 1.拆解手机
  • 2.拆下电池
  • 3.剪下电池上的锂电池保护板

    • 警告:锂电池内部若是破损导致锂接触空气就会冒烟燃烧.小心撕开锂电池保护板和电池主体连接的膜后剪下连接的金属片即可,不要去拆锂电池主体!
  • 4.将法拉电容的正负极接到电池保护板正负极
    • 法拉电容组合型正负极为:引脚长正短负.
  • 5.锂电池保护板装回手机
  • 6.手机充电口接入电源适配器.
  • 7.开机键成功开机.

南瓜电影app投屏到windows

  • 文章作者 SSJ
  • 发布日期 2020年12月21日

有些视频平台只有手机端APP(如韩剧TV),没有电脑端,如果这些app支持视屏投屏功能,那么可以投屏给电脑端的KODI

*优点:KODI投屏接收端没有乐播投屏的广告

  • 到 https://kodi.tv/download 下载PC端KODI后安装
  • KODI > 设置 > 服务 > UPnP/DLNA > 勾选以下两项
    • 启用UPnP支持
    • 允许通过UPnP远程控制(启用UPnP客户端.这将允许你通过一个控制点从任何UPnP服务器获取流媒体并控制播放.
  • 打开南瓜电影或者其他APP播放影片 > 投屏 > 选择Kodi
  • 完.现在电脑上就开始播放影片了.

手机改装无电池工作(5V适配器替代电池)[教程]

  • 文章作者 SSJ
  • 发布日期 2020年11月20日

实践设备:

  • 手机: 红米Note5 (2018年设备)
  • 电源: USB充电器 5V3A (小猫盘充电器)
    • 可以使用任意5V2A以上电源适配器,例如5V2A的UPS或者手机充电器.
    • 5V1A的电源不行,手机自身高负荷运行时若是供电不足会使手机直接掉电.
  • 线材: USB数据线 5V2A (小米支持快充的数据线)
    • USB线不能太细,应当要能支持10W供电通过的USB线.一些赠送的线很可能达不到.若是出现意外掉电情况,检查电源和USB线是否足够承当负载.

步骤

  • 1.拆解手机
  • 2.拆下电池
  • 3.剪下电池上的锂电池保护板

    • 警告:锂电池内部若是破损导致锂接触空气就会冒烟燃烧.小心撕开锂电池保护板和电池主体连接的膜后剪下连接的金属片即可,不要去拆锂电池主体!
  • 4.将USB线的正负极接上电池保护板的正负极
    • 一般红线是+正极, 黑线是-负极, 但也有相反线序的例外情况,为避免反接拿万用表确认正负极是哪根线.
  • 5.锂电池保护板装回手机
  • 6.开机键开机.

拒接电话后提示”您拨打的号码是空号”

  • 文章作者 SSJ
  • 发布日期 2020年9月19日

别人来电,拒绝电话如果对方提示”您拨打的号码是空号”, 这是因为手机开启了[呼叫转移/来电转接]

取消呼叫转移的方法有两种:

  1. 在拨号界面输入##002#再拨打, 即可取消所有呼叫转移, 成功后屏幕会提示操作成功.
  2. 也可以在”骚扰拦截”里面设置”来电转接”( 根据不同品牌手机设置位置有所不同 ).

北京电信ifree使用tasker每月发短信激活本月0月租规则

  • 文章作者 SSJ
  • 发布日期 2020年3月4日

实际最少要消费0.1元.

原理:每月最后一天发送短信102到10001查询已产生费用,若为0.00元则发送一条短信到其他手机.

  • tasker短信匹配内容 *您本月已产生费用为0.00元* ,前后加上*星号匹配.

ifree 0月租规则

  • 每月必须拨出一个通话记录,或者发出一条短信,否则次月会生成5元的”月基本费-手机上网套餐费”.
  • 给10001号的短信和电话,都不激活0元月租.(已客服确认)
  • 接电话和收短信也是无效的.必须主动打出电话或者发出短信.(已客服确认)
  • 对方响铃但没有接通,不算使用记录.必须接通才可以.(已客服确认)

Tasker自动化处理手机短信和通话

  • 文章作者 SSJ
  • 发布日期 2020年2月28日

备用手机需要的自动化处理主要需求:

  1. 短信和未接电话通过网络转发(微信api和邮箱).
  2. 每月自动发短信解除广西钉钉卡的40G流量封顶服务(广西钉”KTBXLL”).
    联通每月1号凌晨25分钟可以成功受理业务.(凌晨05分钟的时候提示无法受理).

IFTTT虽然简单好用并且功能完善(比如自带支持邮件发送,tasker需要额外部署邮件服务),但缺点1是不支持中文内容识别而无法做中文短信内容的识别触发.缺点2是必须要有网络,如果手机卡流量被用完停网,这时候就无法通过预设的短信来解封.

Tasker本地执行并且支持中文字符匹配.

广西钉钉卡短信解除流量限制步骤:

1,给10010发送代码”KTBXLL”(开通不限流量).
2,收到请求确认的短信后再回复”Y”确定来完成解封.

  • tips:给tasker最大权限:允许自启+省电策略无限制+权限管理全部允许+常驻后台.
  • 允许自启动这样手机重启后,Tasker即便首次开机不解锁,也依旧生效可以执行自动计划.
  • 手机做热点宽带的话,设置开机后自动开启热点:tasker> 事件>系统>设备启动(表示手机开机);关联任务>网络>wifi共享>打开.此设置完毕,手机在开机后会自动开启热点wifi.但由于手机并不支持停电后来电自动开机.因此如果遭遇长时间停电还是需要手动操作开机.

Tasker+SendSilentMail v4.52 来实现发送邮件.

用到的几种变量：
%SMSRN (发件人姓名)
%SMSRF (发件人号码)
%SMSRB (短信内容)
%SMSRD (发件日期）
%SMSRT (发件时间）
配置完成后点击【Save Template】，输入配置名后保存。再到Tasker插件中调用配置.

Tasker 通过HTTP Get或者HTTP Post对接微信API

  • WxPusher http://wxpusher.zjiecode.com/ *优选,直接显示需要的消息正文.
  • Server酱-方糖气球 http://sc.ftqq.com/ *很稳定设置里一次后用了两三年(2020.02.28)没有再管程序一直有效发送通知,但是通知只提醒有消息,正文需要点击转跳查看.

安卓端网盘哪个最好用：百度网盘vs天翼云盘

  • 文章作者 SSJ
  • 发布日期 2019年7月30日

假如手机丢失，我们希望恢复的数据：
SSS:相册，拍摄的视频；通讯录，短信；
SS:微信记录，录音，通话录音；
S:通话记录；

  • 前置结论：
    • 百度和天翼的微信备份都仅仅支持通过微信传输的图片，视频，文档，并非聊天记录的转移和恢复。
    • 百度盘的普通会员支持备份自定义的多个目录，适应任意备份的需求。
    • 天翼云不支持备份录音和通话录音。
  • 百度网盘 for Android （190730）
    • 可自定义添加多个目录，并且自动wifi备份。（解决录音和通话录音的备份）√。
    • 可备份：相册，视频；通讯录，短信；通话记录；微信（图片视频文档，不含聊天文本）。
    • 不支持：微信聊天记录备份。
    • 音乐备份功能不包含录音。
  • 天翼云盘 For Android（190730）
    • 不支持：自定义添加目录来备份，因此录音和通话录音无法备份。
    • 可备份：相册，视频；通讯录，短信；通话录音；微信（图片&视频&文档）
    • 音乐备份功能不包含录音。

mailx通过smtp登录QQ邮箱发送邮件

  • 文章作者 SSJ
  • 发布日期 2019年5月29日

最近发现用来申请谷歌教育版的三个教育网邮箱被删号了两个。谷歌盘也被删除，幸好有准备的狡兔三穴，还剩下一个苟活。
基于这种教育邮箱的存活风险率较高，而且平常也并不会登录使用及时的发现是否依旧存活，因此需要一个能够监控邮箱状态是否正常并及时获得提醒的办法。
监控原理：定时通过QQ的SMTP发送邮件给教育网邮箱，如果邮箱不存在了，QQ邮箱会收到错误邮件，从在手机端微信和QQ邮箱得到提示。

系统：centos7

安装 mailx
yum install mailx

测试是否安装成功
which mail
/bin/mail

编辑配置文件
nano /etc/mail.rc
在文件末尾添加

set smtp=smtp.qq.com
set smtp-auth-user=xxx@qq.com

set smtp-auth=login
set smtp-use-starttls
set ssl-verify=ignore
set nss-config-dir=/root/.certs/

因为需要 QQ 邮箱的 SSL 证书，所以我们还需要手动的获取QQ 邮箱的证书保存到本地指定的目录里以备调用和验证，具体命令如下：

mkdir -p /root/.certs/
echo -n | openssl s_client -connect smtp.qq.com:465 | sed -ne '/-BEGIN CERTIFICATE-/,/-END CERTIFICATE-/p' > ~/.certs/qq.crt
certutil -A -n "GeoTrust SSL CA" -t "C,," -d ~/.certs -i ~/.certs/qq.crt
certutil -A -n "GeoTrust Global CA" -t "C,," -d ~/.certs -i ~/.certs/qq.crt
certutil -L -d /root/.certs

为了防止出现发送邮件警告提示，还需要进入邮箱 SSL 证书存放目录
cd /root/.certs 里执行如下命令：

certutil -A -n "GeoTrust SSL CA - G3" -t "Pu,Pu,Pu" -d ./ -i qq.crt

返回如下提示即可：
Notice: Trust flag u is set automatically if the private key is present.
这是为了信任证书的标记操作。

至此，已经完成了 mailx 结合 QQ 邮箱发送系统邮件的部署了。

测试
echo “测试邮件” | mail -s “测试” XXX@qq.com

记账APP对比选择(安卓)

  • 文章作者 SSJ
  • 发布日期 2017年9月11日

记账APP综合排名

第一：随手记

 优点：
可以0元入账，记录某些参考物品价格很有用。
可以在web端编辑处理，功能和app端一样，可上传图片（web端插入的图片大于300KB则被限制失败，这点不如app端方便）。
缺点：
位置信息的商家搜索功能在小城区没数据
开启app时候需要等待广告显示几秒。

第二：微记账

优点：
列表中的记录图片显示比较大，比较容易辨别内容。
开启app时无广告，清爽即时。
缺点：
没有web端。
不可0元入账。
第三：挖财记账

 优点：
每条记录最多可以附加四张图片，适合对消费拍照有特殊爱好的用户。
可在web端处理账本，可查看多张图片（分辨率不高）。
缺点：
web不支持添加图片（但可以查看）。
web图片加载速度慢。比随手记web端图片加载速度慢好几倍。
第四：一笔记账
优点：
界面清爽，简单。
无广告。
缺点：
太简单……不支持插入图片。
没有搜索功能。
没有位置信息功能。
记账只能记录三个信息：金额、备注、分类
