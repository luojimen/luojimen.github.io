南瓜电影app投屏到windows

  • 文章作者 SSJ
  • 发布日期 2020年12月21日

有些视频平台只有手机端APP(如韩剧TV),没有电脑端,如果这些app支持视屏投屏功能,那么可以投屏给电脑端的KODI

*优点:KODI投屏接收端没有乐播投屏的广告

  • 到 https://kodi.tv/download 下载PC端KODI后安装
  • KODI > 设置 > 服务 > UPnP/DLNA > 勾选以下两项
    • 启用UPnP支持
    • 允许通过UPnP远程控制(启用UPnP客户端.这将允许你通过一个控制点从任何UPnP服务器获取流媒体并控制播放.
  • 打开南瓜电影或者其他APP播放影片 > 投屏 > 选择Kodi
  • 完.现在电脑上就开始播放影片了.
  • 标签 下载, 开始, 手机, 电脑, 视频

rclone批量编辑谷歌共享盘

  • 文章作者 SSJ
  • 发布日期 2020年5月28日

一个账号下可以拥有多个谷歌共享云端硬盘,当这些共享盘需要批量添加到rclone中时,可通过配置文件复用修改,只需要修改’ team_drive = ‘参数后的团队盘编码

  • 团队盘编码获取
    • 浏览器打开团队盘,地址栏显示的最后一段便是团队盘的固定编码:示例https://drive.google.com/drive/u/0/folders/baMuWbweYtN6Uksdf的’ baMuWbweYtN6Uksdf ‘便是团队盘编码.
  • 编辑配置文件
    • 查看配置文件位置 ‘ rclone config file ‘
    • 编辑配置文件 nano ~/.config/rclone/rclone.conf
    • 复制黏贴一份已经正确的配置
    • 修改[名称]和其中的 team_drive = ‘团队盘编码’
  • 备份配置文件
    • rclone配置文件存储位置~/.config/rclone/rclone.conf，可以保存里面的内容，在新电脑上直接用这个配置文件即可恢复使用.

神舟战神Z7M-KP7GZ笔记本windows10蓝屏故障解决

  • 文章作者 SSJ
  • 发布日期 2019年8月6日

这三四天来每日笔记本都会出现一次蓝屏自动重启；今天笔记本运行一晚上，早上醒来还正常，打开QQ使用一段时间胡，出现蓝屏，提示：FAULTY HARDWARE CORRUPTED PAGE。此蓝屏错误很可能与硬件内存错误或系统文件损坏有关。

三个可能：
1、内存故障
2、快速启动的故障
3、驱动故障

解决办法：

1、检测内存

  1. 运行 C:\Windows\system32>sfc /scannow 开始系统扫描。
  2. CMD运行运行windows内存诊断：mdsched.exe，重启后开始检测内存，检测完自动重启进入系统，提示“没有检测到内存错误”。

2、关闭快速启动

此蓝屏错误很可能与硬件内存错误或系统文件损坏有关。 而快速启动意味着你上次的关机并不是完全关机。
快速启动原理和休眠类似，但是所有用户进程（比如你开的记事本，浏览器之类的）都会被结束掉。结束后，内存里就剩下内核及系统相关的模块，还有一部分驱动了，这时候，把它们写到硬盘里的一个文件里，下次开机直接把它们读进来就好了，所以你就会觉得开机速度很快。

  1. 登陆系统，点击左下角开始菜单。
  2. 点击设置按钮，进入Windows设置界面。
  3. 点击系统按钮，进入系统设置界面。
  4. 点击电源和按钮，进入电源和睡眠设置界面。
  5. 点击其他电源设置，进入电源选项界面。
  6. 点击选择电源按钮的功能，进入电源选项下的系统设置界面。
  7. 点击更改当前不可用的设置，使能关机设置的选项。
  8. 将启用快速启动前面的勾取消，点击保存修改即可。
  9. 重启电脑。

3、重新安装一遍官方驱动

  1. 1Chipset 芯片组驱动
  2. 2Intel VGA 集显驱动
  3. 4Card reader 读卡器驱动
  4. 5LAN 网卡驱动
  5. 6Intel Wifi_BT 无线网卡及蓝牙驱动
  6. 7Intel ME ME驱动
  7. Serial IO驱动 Serial IO
    重启电脑

经过以上3个处理后，连续开机两天，打开各种应用和游戏，没有再出现蓝屏，故障解决。

Link Shell Extension 让网盘同步不同的文件夹

  • 文章作者 SSJ
  • 发布日期 2018年8月14日
推荐一个mklink管理软件，让只能同步单个指定目录的onedrive，google drive等网盘客户端可以同步不同的文件夹。
Link Shell Extension （有中文语言）
http://schinagl.priv.at/nt/hardlinkshellext/hardlinkshellext.html
原理是使用mklink，Link Shell Extension是mklink的图形软件。

Windows下mklink使用,   硬链接, 软链接和快捷方式的区别：

符号链接（Symbolic link）

  • 执行命令 mklink link_name target_name
  • 创建链接后的图标和快捷方式很像, 都有一个箭头的标志
  • 在系统中不占用空间
  • 在文件系统中不是一个单独的文件
  • 在操作系统层解析（！？）
  • 如果源文件被删除了，链接就没用了
  • 移除源文件不会影响符号链接
  • 移除符号链接文件也不会影响源文件
  • win10_x64_build10565上测试不可以右键修改图标和设置管理员运行
  • 文件大小为0字节和不占用空间
  • 文件属性的创建时间和修改时间都是软链接创建和修改时的时间
  • 文件类型是.SYMLINK

硬链接（Hard link）

  • 执行命令 mklink /H link_name target_name
  • 在系统中占用的空间与源文件相同，但在系统中引用的是相同的对象（不是拷贝）
  • 在操作系统层解析（！？）
  • 图标和创建快捷方式的图标不同(没有快捷方式的小箭头)
  • 移除源文件不会影响硬链接
  • 移除硬链接不会影响源文件
  • 如果源文件被删除，它的内容依然通过硬链接存在
  • 硬链接文件的任何更改都会影响到源文件
  • 文件大小, 占用空间, 创建和修改时间跟原原文件一样
  • 可以在cmd下运行硬连接(假如链接的是程序)

快捷方式（Shortcut）

  • 在选择的源文件上鼠标右键，通过下拉菜单创建
  • 快捷方式在系统中跟源文件是完全分离的
  • 只有那些懂得快捷方式的程序知道它们
  • 如果源文件删除，链接就没用了
  • 移除源文件不会移除快捷方式
  • 移除快捷方式不会影响到源文件
  • 可以右键更改图标或者设置管理员运行
  • 文件属性的创建时间和修改时间都是快捷方式创建和修改时的时间
  • 文件大小仅有几百字节, 跟原文件大小无关
  • 文件类型是.lnk
  • 可以在cmd下运行快捷方式(假如链接的是程序, 且运行命令是XXX.LNK)(win10_x64_build10565上测试通过)

  • 标签 google drive, 电脑
